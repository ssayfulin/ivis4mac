======  Using MySQL Workbench to modify model ========= 

1. Start MySQL Workbench
2. Create New Model through menu 
     File/New Model
3. Import data from poiskdetey.sql script  
    File/Import/Reverse Engineer MySQL Create Script

    Select sql file. Check "Place imported objects on diagram"  Checkbox.

4. There should be pile of tables  

   Select Arrange/AutoLayout menu item to have more nice view.
   Change Model/Diagram Properties and Size if required

5. Do modifications if necessary in MySQL Workbench

6. Export changes back to sql

   File/Export/Forward Engineer SQL Create script

7. Check changes with diff tool
